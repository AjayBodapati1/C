**Tic-Tac-Toe:**:
-----------------
This is a simple console-based Tic-Tac-Toe game written in C. It allows a human player to play against the computer. The game follows the traditional rules of Tic-Tac-Toe, where player take turns marking the spaces in a 3Ã—3 grid with respective symbols ('X' for the player and 'O' for the computer). The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

**Features**:
------------
Single-player mode against the computer
Randomized computer moves
User-friendly console interface
Option to play multiple rounds
